# Security Policy

## Scope

This project is a collection of text prompts. There is no executable code, no dependencies, and no data collection.

Security vulnerabilities in the traditional sense do not apply here. However, if you discover that a prompt in this collection:

- Can be used to bypass AI safety guidelines
- Encourages harmful, deceptive, or malicious use of AI
- Produces outputs that could cause real-world harm

Please report it responsibly.

## Reporting

Open a GitHub issue with the label `security` and describe the concern. Do not include harmful prompt outputs in public issues — describe the problem without reproducing it.

Reports will be reviewed within 7 days.
